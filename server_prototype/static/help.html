What Do I Do?

Welcome, Commander. In this game, you're one of an elite team sent to colonize Mars—but something's gone wrong. Trapped inside one of the auxiliary facilities on the base, your only hope is to command the fleet of advanced robots, repair the base, and discover what happened.

Mission Control is always by your side. At the start of each level, they'll let you know what needs to be done. You can always see your objectives under the map on the left. Write your code in the editor on the right—assembling blocks or writing Python—then run it using the button on the bottom left. Mission Control will monitor your progress and show you what you've accomplished.

Block Types

The Tell Block

The tell block gives a command to an object. The command is one of the object's methods, or a method from one of the superclasses of the object's class. That is, an object knows how to do anything from its class, and it knows how to do anything from a class that is a parent of its class.

The Object Block

An object is something that our computer systems can interact with. Each object is based on a template called a class. This template controls what each object knows how to do. You can issue commands to it via the tell block.

The Method Block

Each object knows how to do certain things. The method block represents something an object of a particular class can do. Use the tell block to issue the command to the object.
